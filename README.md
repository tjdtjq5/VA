# Project Path

모바일 / 한붓그리기 캐주얼 게임
Unity 기반으로 개발

---

## 🎮 프로젝트 개요
- **장르**: 캐주얼
- **기간**: 2025.06 ~ 2025.09
- **참여 인원**: 2명 (저 포함)

---

## 📌 핵심 기능
- **전투 시스템:** Animator 기반 전투 구조 / 상태 관리 / AI 이동
- **UI 시스템:** 바인딩 자동화 툴, 메뉴 전환, 인게임 UI 구조
- **툴 제작:** ScriptableObject 기반 성장 시스템, 커스텀 에디터 툴
- **클라이언트 최적화:** 빌드 자동화, Addressable 기반 리소스 관리
- **서버 연동:** ASP.NET Core 서버와 연동된 로그인 및 데이터 저장 기능

---
